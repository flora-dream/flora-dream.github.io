项目介绍

项目学习地址

本项目是Python + Flask + Echarts打造的疫情监控系统，涉及的技术有：

  • Python 网络爬虫
  • 使用 Python与MySQL数据库交互
  • 使用Flask构建Web项目
  • 基于Echarts数据可视化展示
  • 在Linux上部署web项目及爬虫

项目架构

数据获取 –> 数据持久化 –> flask搭建web后台 –> 数据可视化 h5 + echarts

项目环境

  • Python 3.6
  • MySQL 5.7
  • PyCharm（Python IDE）
  • Jupyter notebook（Python IDE）
  • HBuild （前端IDE）
  • Linux主机 （后期项目部署）

修改Jupyter工作目录

  1. 打开cmd输入jupyter notebook --generate-config创建配置文件，可以看到生成在C:\Users\66\.jupyter\jupyter_notebook_config.py目录下

  1. 打开该文件搜索notebook_dir并取消改行注释，将路径修改为想要存放的工作路径，并且添加r让\不被转义

  2. 工作路径下新建记事本并修改成start.cmd，在该记事本中写入jupyter notebook，以后双击该文件变自动打开Jupyter

notebook常用快捷键

  • 单元格类型转换：Y、M
  • 插入单元格：A、B
  • 进入命令模式：Esc
  • 运行单元格： ctrl / shift / alt + enter
  • 删除单元格： DD
